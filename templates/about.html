About This Project

What is this?

Smart Expense Tracker is a web application that helps you track your daily expenses and uses Machine Learning to predict your future spending patterns.

Features

  • Add, edit, and delete expenses
  • Categorize spending (Food, Transport, etc.)
  • View analytics with charts and graphs
  • ML-based spending prediction
  • Anomaly detection for unusual spending

Tech Stack

  • Python + Flask (Backend)
  • HTML/CSS/JS (Frontend)
  • SQLite (Database)
  • Pandas + Matplotlib (Analytics)
  • Scikit-learn (Machine Learning)
